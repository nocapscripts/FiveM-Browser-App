# FiveM-Browser 🖥️

**Test Release Android** – In-game GTA servers browser app for FiveM, now available for testing.

![FiveM Browser About](https://i.imgur.com/lHv6pZt.jpeg)
![FiveM Browser Main](https://i.imgur.com/1bxuHmI.jpeg)

## Description
FiveM-Browser is a lightweight in-game browser designed for FiveM servers. This repository hosts the **test release** for Android devices, intended for developers and testers to explore functionality and provide feedback.  

> ⚠️ **Note:** This is a test release. Features may be unstable, incomplete, or behave differently on various devices.

## Features
- Full in-game browser support for FiveM
- Simple, intuitive interface
- Lightweight and fast
- Supports browsing websites, server resources, and in-game overlays
- Designed for testing and development purposes

## Supported Devices
| Device Type | Supported | Notes |
|-------------|-----------|-------|
| Android Phones | ✅ Yes | Android 8.0+ recommended |
| Android Tablets | ✅ Yes | Tested on major models |
| iOS Devices | ❌ No | Not supported in this release |
| PC/Emulator | ⚠️ Limited | May work with some emulators |

> Devices not listed above may experience instability or incomplete functionality.

## App States
| State | Description |
|-------|-------------|
| Loading | Initial startup of the browser inside the game |
| Ready | Browser is fully loaded and interactive |
| Offline | No internet connection or server unavailable |
| Error | Browser failed to load content. Report bugs here: [GitHub Issues](https://github.com/nocapscripts/FiveM-Browser/issues) |
